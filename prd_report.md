# تقرير متطلبات المنتج (PRD) وحالة المشروع الحالية
**المشروع:** AI Image Studio (Zero-Cost Architecture)
**تاريخ التقرير:** 26 يونيو 2026

---

## 1. نظرة عامة (Overview)
الهدف من المشروع هو بناء منصة أدوات ذكاء اصطناعي للصور (AI Image Tools) تعمل باحترافية وتعتمد على بنية تحتية مجانية (Zero-Cost) لتقليل المصاريف التشغيلية (0$). تم التحول من خدمات مدفوعة (مثل Clipdrop) إلى استخدام النماذج المجانية عبر **Hugging Face Inference API**، مع بناء لوحة تحكم إدارية (Admin Dashboard) للتحكم الدقيق في كافة أدوات ونماذج الموقع، بالإضافة إلى تفعيل نظام قواعد البيانات (Supabase) لحفظ الإعدادات بأمان.

---

## 2. أين نحن الآن؟ (Current Status)
المشروع الآن في **مراحله شبه النهائية**. تم إيقاف العمليات مؤقتاً بناءً على طلبك (وبسبب وصول فريق الذكاء الاصطناعي للحد الأقصى من حصة التشغيل - Quota). 
تم إنجاز البنية التحتية بالكامل، ولوحة التحكم، وربط معظم أدوات التوليد. يتبقى فقط اللمسات النهائية على الواجهة الأمامية لأداة إزالة الخلفية.

---

## 3. ما تم إنجازه بالكامل (Completed Milestones)

### المرحلة الأولى: البنية التحتية للإعدادات (Settings Schema & API Refactor) ✅
- تم إنشاء جدول `app_settings` للعمل مع قاعدة بيانات **Supabase**.
- تمت برمجة نظام يعتمد على الذاكرة المؤقتة (Mock Database) لضمان عمل الموقع حتى قبل ربط قاعدة البيانات الحقيقية من قِبل المستخدم.
- تم تحديث 5 أدوات للعمل من خلال قراءة المفتاح (`HUGGINGFACE_API_KEY`) من الإعدادات بدلاً من الأكواد الثابتة:
  - Text-to-Image (تحويل النص لصورة)
  - Cleanup (إزالة الشوائب - Inpainting)
  - Upscale (رفع الدقة - Super Resolution)
  - Reimagine (إعادة التخيل - Variations)
  - Uncrop (توسيع الصورة - Outpainting)

### المرحلة الثانية: لوحة تحكم الإدارة (Advanced Admin Dashboard) ✅
- تم بناء واجهة متطورة (Premium UI) محمية بكلمة مرور.
- إضافة إمكانية التحكم في **مفاتيح API** الخاصة بـ Hugging Face.
- إضافة إمكانية التحكم في **أسماء النماذج (Models)** لكل أداة على حدة.
- **الإعدادات المتقدمة (Advanced AI Settings):**
  - التحكم في `Inference Steps` (عدد خطوات التوليد لزيادة الدقة).
  - التحكم في `CFG Scale` (مدى التزام النموذج بالنص).
  - التحكم في `Negative Prompt` (استبعاد التشوهات).
  - التحكم في أبعاد المخرجات `Width` و `Height`.
- **حدود الاستخدام (Site Controls):** التحكم في `rate_limit_free` (عدد المحاولات المجانية المسموحة لكل مستخدم).

## 4. ما هو قيد التنفيذ / متبقي (Pending Tasks)

### المرحلة الثالثة: أدوات الخلفية (Background Tools Refactor) ⏳
- **ما تم إنجازه منها:** 
  - تم إلغاء مكتبة المتصفح (WASM) التي كانت تسبب خطأ `removeBackground is not a function`.
  - تم بناء مسار API خلفي جديد (`/api/remove-background`) ليرتبط بنموذج `briaai/RMBG-1.4` عبر Hugging Face.
- **ما تبقى منها:**
  - إكمال ربط واجهة المستخدم (Frontend) بمسار الـ API الجديد لصفحتي (`remove-background` و `replace-background`).
  - تشغيل الاختبارات النهائية (E2E Tests) للتأكد من سلاسة رفع الصورة وعودتها مفرغة.

---

## 5. المتطلبات التقنية الحالية (Technical Specs)
- **إطار العمل (Framework):** Next.js 14 (App Router).
- **قاعدة البيانات:** Supabase (PostgreSQL) لجدول `app_settings` وجدول الاستخدام.
- **موفرو الذكاء الاصطناعي:** Hugging Face Inference API (مجانًا).
- **الاستضافة المتوقعة:** Vercel.
- **التصميم:** Tailwind CSS + UI متقدم (Glassmorphism, Dark Theme).
